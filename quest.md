# 2주차 퀘스트

## 변경 이력

1 - 유지 : 제작 의도와 수행 내용이 좋은 방향이라고 판단했다.

2 - 삭제 : 미션을 수행하면서 반드시 수행해야 할 내용이기 때문에 미션과 중복된다고 판단했다.

3 - 수정 : 체크리스트 선정은 미션 필수 요소이므로 체크리스트 보고 잘 만든 체크리스트인지 분석하는 방향으로 발전했다.

4 - 수정 : 질문 하나는 난이도가 너무 쉬워서, 3주차 그룹의 gist에 comment를 남기는 방향으로 수정했다.

5 - 삭제 : 미션을 수행하면서 당연하게 수행할 수 있는 목표라고 판단했다.

6 - 수정 : 공식 문서의 범위가 너무 낮기 때문에 라이브러리에서 제공하는 메서드 하나를 분석하는 방향으로 수정했다.
